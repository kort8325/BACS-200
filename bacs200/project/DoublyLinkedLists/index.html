Doubly Linked Lists

Linked lists are a series of nodes that link between eachother as a method of traversing the list, and a doubly linked list is a traversable in both directions. One node would link to the next, and the last node will link to the first node and the first one would connect to the last one. A Node is contructed and put into a doubly linked list by connecting the node to one node ahed and one node behind it. Advantages of a doubly linked list is how easy it is to traverse. The ability to traverse in both directions makes doubly linked lists easy to manuver and quick to run through, and the methods can be much faster than the same methods in an array, along with doubly linked lists using multivariable types.
