Python, the simple language

Work on python began in the late 1980's by Guido van Rossum, who had worked on google for a time.It became the main coding language used by the Netherlands, succeeding the ABC language and was notable because of it's ability of exception handling and interfacing, which would become a standard for languages in the future. Today, Python 3.7 is the language that various software development developers default to, because of the simple coding of complex functions and methods.

One of the huge reasons, python is used as often as it is, is the way python 3.7 implements various lists to give the best running time possible. Running time means the amount of time needed for a function to be completed, and with huge lists or databases of values, having a quicker running time is always better for programmers.
