Deques

Deques are a combination of the first in first out implementatin of queues, and the last in first out implementation of Stacks, using both to make a list more manuverable. A deque is a "double ended queue," using the same methods from both queues and stacks. Advanages to using a deque over either stacks or queues are apparent, but there are also positives to usinga deque over an array in python, such as the run time. The advantages to runtime stay consistant from stacks and queues, making a deque's methods extreamly fast, but missing the traversability of a usual array or linked list.
