Singly Linked Lists

Linked lists are a series of nodes that link between eachother as a method of traversing the list, and singly linked lists are specifically only linked in one direction. A node is constructed and will contain a data variable, and will be linked to the next node. The nodes theoretically will start at a first node and will link together until they will hit an ending node, and the end node will be linked up to the first node a gain, making the linked list a loop for searching through. The advantages of using a linked list for storing values are that the lists are traversable and are more easily accessable than the default array usually is in python, and the nodes are a general data type of none allowing the list to be multivariable.
