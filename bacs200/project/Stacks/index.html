Stacks

Stacks are an extreamly fast and simple implementation of a list, using a last in first out approach to a list. As the name implies, a stack is like a stack of books, where you can only add or remove a book from the top of the stack. The implimentation of a stack makes the methods of it extremly quick, but it is because of the limitation to the operablility of the list. A stack is not always the best option for a list, however if you only need to use the elements at the top of a stack, then it is a great option for implementing a list in python.

class Stack:
    def __init__(self):
        self.stack = []
        self.top = None
  
    def push(self, data):
        self.stack.append(data)
        self.top = data
  
    def pop(self):
        temp1 = self.stack.pop()
        temp = ""
        for i in self.stack:
            if(i == temp1):
                break
            temp = i
        self.top = temp
        return(temp1)
  
    def peek(self):
        return(self.top)
  
    def isEmpty(self):
        if(self.top is None):
            return True
        return False
  
    def size(self):
        counter = 0
        for i in self.stack:
            counter = counter + 1
        return(counter)
