Stacks

Queues are an extreamly fast and simple implementation of a list, using a First in first out approach to a list. A Queue is just like a stack, but instead of a last in first out approach, a queue uses a first in last out approach, only allowing operations to show the first item on the queue. The implimentation of a Queue makes the methods of it extremly quick, but it is because of the limitation to the operablility of the list. A queue is not always the best option for a list, however if you only need to use the elements at the front of a queue, then it is a great option for implementing a list in python.

class Queue:
    def __init__(self):
        self.back = None
        self.queue = []
  
    def enqueue(self,data):
        self.queue.append(data)
        if self.back is None:
            self.back = data
  
    def dequeue(self):
        pop = self.queue[0]
        temp = []
        var = 1
        while(var<=len(self.queue)-1)
            temp.append(self.queue[var])
            var = var +1
        self.queue = temp
        self.back = self.queue[0]
        return pop
  
    def isEmpty(self):
        if self.back is None:
            return True
        else:
            return False
  
    def size(self):
        counter = 0
        for i in self.queue:
            counter = counter + 1
        return counter
