Hash Lists

Hash lists differ from pythong lists in the way they store data, and how the data is accessed. Using the modulo function, a Hashlist will find the remainder of the data inputed from the length of the list, and will store the data based on that remainder. Modulo is used in pythong with the % key, and will output a remainder for a hashlist. When the data is stored based on the remainder of the data modulo the the length of the list, then searching for a data value will be easier, and quicker to process. The advantages of a Hash list are not just that they are faster for accessing data than an array, but they also are quicker than arrays at traversing a list.

class HashList:
    def __init__(self, length):
        self.hlist = [None] * length
        self.len = length
  
    def print(self):
        print(self.hlist)
  
    def hashfunction(self, item):
        return item % self.len
  
    def rehashfunction(self, oldpos):
        return (oldpos + 1) % self.len
  
    def contains(self, item):
        for i in self.hlist:
            if (i==item):
                return True
        return False
  
    def put(self, item):
        hashvalue = self.hashfunction(item)
        if(self.hlist[hashvalue] == None):
            self.hlist[hashvalue] = item
        elif(self.hlist[hashvalue] != None):
            return (self.put2(hashvalue+1,item,hashvalue,0)
  
    def put2(self, hashvalue, item, origin, counter):
        if(self.hlist[hashvalue] == None):
            self.hlist[hashvalue] = item
        elif(self.hlist[hashvalue] != None):
            if hashvalue==origin:
                return print("error")
            if hashvalue == self.len-1:
                return (self.put2(0,item,origin,counter))
            else:
                return (self.put2(hashvalue + 1,item,origin,counter))
  
    def items(self):
        string = ""
        for i in self.hlist:
            if i != None:
                string += str(i) + ", "
        return string
